Iowa Demographics Analysis (STAT 220)

R Google Sites Data Visualization Web Presentation

Project Overview

For STAT 220, this project went beyond simple data analysis by incorporating web presentation. A dedicated Google Sites website was created to document and display an in-depth statistical analysis of Iowa demographic data. The goal was to present the entire analytical journey, from data sourcing to final conclusions, in an accessible and comprehensive online format.

Methodology

  • **Data Sourcing:** Identified and acquired a public dataset related to Iowa demographics.
  • **Data Analysis in R:** Performed data cleaning, exploratory data analysis, and created key visualizations in R.
  • **Content Creation:** Wrote descriptive text explaining the project's purpose, the methodology used, and the interpretation of the results.
  • **Website Development:** Used Google Sites to structure the content logically, embedding visualizations and text to create a narrative-driven project showcase.

Key Feature: Project Website

The main deliverable was the website itself, which served as a dynamic report. This approach allowed for a richer presentation of findings than a static document would permit.

The site was structured with distinct pages for the introduction, methodology, key findings, and conclusion, making the complex analysis easy for any visitor to follow.

Conclusion & Impact

This project demonstrated the importance of effective science communication. By using a web platform, the statistical findings were made more engaging and accessible to a wider audience. It was an excellent exercise in translating technical analysis into a clear and compelling story, a crucial skill in the field of data intelligence.
