Code Snippet (R)

An example of a custom function written for this project is shown below. This function was created to automate the process of checking model residuals for normality. 


# Custom function to check normality of residuals
check_normality <- function(model) {
  residuals <- residuals(model)
  
  # Shapiro-Wilk test for normality
  shapiro_test <- shapiro.test(residuals)
  
  # Q-Q plot
  qqnorm(residuals)
  qqline(residuals, col = "#03DAC6")
  
  return(shapiro_test)
}
