Data Visualization Critique & Redesign (STAT 324)

R ggplot2 Data Visualization Statistical Critique

Project Overview

This project was a comprehensive exercise in **data visualization principles**, focusing on **William Playfairâ€™s historical chart**, *"Extent, Population, and Revenues of the Principal Nations in Europe."* The objective was twofold: to **critique the original graphic's effectiveness** and to **re-design and re-code** the visualization using **R and ggplot2** to enhance clarity, precision, and adherence to modern design principles.

The re-design centered on correcting the perceptual flaws of the original chart, specifically replacing area-based encoding with length-based encoding, which is perceptually superior for quantitative comparison.

Methodology and Re-design Principles

  • **Critique:** Identified the original chart's strengths (hierarchy, color contrast) and major flaws (inaccurate comparison due to using **circle area** to represent 'Extent').
  • **Aesthetic Mapping:** Replaced the low-precision circle area with **bar length (height)** for all three variables (Extent, Population, and Taxation) to maximize accuracy.
  • **Scaling:** Applied a **pseudo-log transformation** to the Extent (Area) data to manage the wide range of values effectively, ensuring both large and small countries were visible and comparable.
  • **Execution:** Used **R** with the **tidyverse** and **ggplot2** packages to execute the full re-coding and design.

Code Snippet: Redesign Implementation (R/ggplot2)

The core of the re-design involved mapping all three variables to height and utilizing a non-linear scale for the area data. 


library(tidyverse)
library(scales) 

europe <- read.csv("playfair_european_nations.csv")
europe$Country <- fct_reorder(europe$Country, europe$Area, .desc = TRUE)

# Scaling Area data and applying pseudo-log transformation on the y-axis
europe <- europe %>%
  mutate(Area_scaled = Area / 10)

ggplot(europe) +
  # 1. Main Bar for Extent (Area_scaled)
  geom_rect(aes(xmin = as.numeric(Country) * 8 - 2.5,
                xmax = as.numeric(Country) * 8 + 2.5,
                ymin = 0, 
                ymax = Area_scaled,
                fill = as.factor(Power)),
            color = "black", alpha = 0.7) +
  
  # 2. Left Bar for Population (Red)
  geom_rect(aes(xmin = as.numeric(Country) * 8 - 2.0,
                xmax = as.numeric(Country) * 8 - 1.0,
                ymin = 0, 
                ymax = Population),
            fill = "red") +
            
  # 3. Right Bar for Taxation (Goldenrod)
  geom_rect(aes(xmin = as.numeric(Country) * 8 + 1.0,
                xmax = as.numeric(Country) * 8 + 2.0,
                ymin = 0, 
                ymax = Taxation),
            fill = "goldenrod") +
  
  # 4. Dotted connecting segment for Population vs. Taxation
  geom_segment(aes(x = as.numeric(Country) * 8 - 1.5, 
                   xend = as.numeric(Country) * 8 + 1.5,
                   y = Population, yend = Taxation),
               color = "black", linetype = "dotted") +

  # Set scales and themes for clear communication
  scale_y_continuous(trans = pseudo_log_trans(sigma = 1),
                     labels = label_scientific()) +
  scale_x_continuous(breaks = (1:12) * 8, labels = europe$Country) +
  labs(title = "Redesigned Chart of European Nations (Length-based Encoding)",
       x = "Name of Countries", 
       y = "Extent (Pseudo-log Scale)", 
       fill = "Relative Power") +
  theme_minimal()

Conclusion & Impact

The redesigned chart successfully rectified the perceptual flaws of the original, demonstrating the principle that **length (bar height) is a superior aesthetic mapping to area (circle size)** for comparing quantitative values. By utilizing consistent spacing, clear color separation, and a pseudo-log scale, the visualization offers a precise and highly interpretable view of the data.

**Significance:** The new visualization clearly supports Playfairâ€™s historical argument that **England was taxed more heavily than other European nations** by allowing for a precise comparison of the Population and Taxation bars across all countries.
