Statistical Survey Analysis (STAT 310)

R Survey Data Data Cleaning Descriptive Statistics

Project Overview

For STAT 310, this project involved a comprehensive analysis of a survey dataset. The objective was to practice the full lifecycle of survey data analysis, from initial data handling to the final interpretation of results. Working with survey data presents unique challenges, including missing values and varied data types, which were central to this project.

Methodology

  • **Data Import & Cleaning:** Loaded the survey data into R and performed extensive cleaning, including handling missing data (NA values), correcting data types, and recoding variables.
  • **Descriptive Statistics:** Calculated summary statistics (mean, median, standard deviation) for quantitative variables and frequency distributions for categorical variables to get an initial understanding of the data.
  • **Data Visualization:** Created visualizations such as histograms, bar charts, and box plots using `ggplot2` to illustrate the distribution of responses and relationships between variables.
  • **Inferential Analysis:** Applied appropriate statistical models to identify significant trends and patterns within the survey responses.

Code Snippet (R)

A key part of the project was visualizing the distribution of a categorical survey question. The following R code using `ggplot2` demonstrates this. 


# Load the ggplot2 library
library(ggplot2)

# Create a bar chart for a categorical variable from the survey
ggplot(survey_data, aes(x = factor(satisfaction_level))) +
  geom_bar(fill = "#03DAC6", color = "#121212") +
  labs(title = "Distribution of Customer Satisfaction Levels",
       x = "Satisfaction Level (1-5)",
       y = "Number of Respondents") +
  theme_minimal()
