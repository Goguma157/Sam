ANOVA Application (STAT 330)

R ANOVA Hypothesis Testing

Project Overview

This project for STAT 330 involved applying the Analysis of Variance (ANOVA) statistical test to determine whether there were any statistically significant differences between the means of two or more independent groups. The focus was on the end-to-end process of hypothesis testing.

Methodology

  • **Hypothesis Formulation:** Defined a null hypothesis (H₀) stating no difference between group means and an alternative hypothesis (H₁) stating at least one group mean is different.
  • **Data Preparation:** Loaded the dataset into R, ensuring it was clean and structured correctly for the ANOVA test.
  • **Assumption Checks:** Verified key assumptions for ANOVA, such as normality of data and homogeneity of variances.
  • **ANOVA Test Execution:** Utilized R's built-in `aov()` function to perform the analysis.

Code Snippet (R)

The core of the analysis was performed using the following R code, which models the relationship between a quantitative dependent variable and a categorical independent variable. 


# Load necessary libraries
library(ggplot2)

# Load the dataset
data <- read.csv("your_dataset.csv")

# Perform the ANOVA test
# 'dependent_variable' is numeric, 'group_variable' is a factor
anova_result <- aov(dependent_variable ~ group_variable, data = data)

# Display the summary of the test
summary(anova_result)

# Post-hoc test (if ANOVA is significant) to see which groups differ
TukeyHSD(anova_result)

Conclusion & Interpretation

The ANOVA test yielded a p-value of [Your P-Value, e.g., 0.02], which was less than the significance level of 0.05. Consequently, we rejected the null hypothesis, concluding that there is a statistically significant difference between the group means. Post-hoc analysis using Tukey's HSD test further identified which specific groups differed from one another. This project solidified my practical understanding of hypothesis testing in R.
