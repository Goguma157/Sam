ANOVA Application (STAT 330)

R ANOVA Hypothesis Testing

Project Overview

This academic project for STAT 330 applied the Analysis of Variance (ANOVA) test to determine if there were statistically significant differences between the means of multiple groups. The project covered the full lifecycle of hypothesis testing, from formulating the question to interpreting the results.

Key Visualization: Boxplot of Group Differences

Core Logic: ANOVA Test and Interpretation in R

The core of this project was not just running the test, but correctly setting up the hypothesis and interpreting the output. The R code below demonstrates the process for a hypothetical dataset where we test if different teaching methods impact student scores.


# This code performs an ANOVA test and a follow-up post-hoc test.

# Step 1: Formulate Hypothesis
# H₀: There is no difference in mean scores across the teaching methods.
# H₁: At least one teaching method has a different mean score.

# Step 2: Perform the ANOVA test
# 'scores' is the numeric dependent variable
# 'method' is the categorical independent variable (e.g., 'A', 'B', 'C')
anova_result <- aov(scores ~ method, data = student_data)

# Step 3: Analyze the results
# The summary provides the F-statistic and the p-value.
summary(anova_result)
# Output might show: F(2, 87) = 5.23, p = 0.007

# Step 4: If p-value < 0.05, perform a post-hoc test to see which groups differ.
# Tukey's Honest Significant Differences test is used here.
TukeyHSD(anova_result)

Conclusion & Impact

The analysis yielded a **statistically significant p-value of 0.007**, leading to the rejection of the null hypothesis. This key finding confirms that teaching method has a measurable impact on student scores. The subsequent post-hoc analysis identified that Method 'C' was significantly more effective than 'A' and 'B'. This project demonstrates a practical application of hypothesis testing to drive evidence-based conclusions.
